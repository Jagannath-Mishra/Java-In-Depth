# Logical Operators

Logical operator are used to test multipel condition. 

e.g. 
```
if(a>b && b>c){
  do something
}
```

## Type of Logical operator
  
  **Operator ------------- Name--------------e.g. (let x = 5, y = 6)**
 
 &&----------------------AND------------------((x<7) && (y<3)) --> false
 
 ||----------------------OR------------------((x<7) || (y<3)) --> ture
 
 !----------------------NOT------------------!(x<7) --> false
